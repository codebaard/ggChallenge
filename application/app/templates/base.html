Goodgame Coding Challenge

call /troops?[query] to get a random distribution of troops

possible query parameters are:

?amount=X, where X denotes the size of the desired army, defaults to 100. Only integers between 3 and 65535 allowed.

?dist=Y, where Y denotes the algorithm which determines the probability distribution. Allowed values are gaussian or flat. Defaults to gaussian.

An example query might look like: /troops?amount=4711&dist=flat
